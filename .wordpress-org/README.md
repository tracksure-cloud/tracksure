# WordPress.org Assets

This directory contains assets for the WordPress.org plugin directory listing.

## Required Assets

### Plugin Icon

- `icon-128x128.png` - Small icon (128x128px)
- `icon-256x256.png` - Large icon (256x256px)
- Or `icon.svg` - SVG icon (recommended)

### Plugin Banner

- `banner-772x250.png` - Low resolution banner
- `banner-1544x500.png` - High resolution banner (Retina)

### Screenshots

- `screenshot-1.png` - First screenshot
- `screenshot-2.png` - Second screenshot
- etc.

## Current Status

⚠️ No assets uploaded yet. Add your plugin icon and banner images here.

## Image Guidelines

- Icons: Square, simple, recognizable at small sizes
- Banners: 772x250px (normal) and 1544x500px (Retina)
- Screenshots: PNG or JPG, max 1MB each
- Use plugin name/logo, avoid text clutter

## Upload Process

When deploying via GitHub Actions, these assets will be automatically uploaded to WordPress.org SVN in the `assets/` directory.
