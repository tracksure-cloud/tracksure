## 🚀 **Getting Started**

### **1. For Plugin Users**

```bash
# Install from WordPress.org
Visit: Plugins → Add New → Search "TrackSure"
Or download: https://wordpress.org/plugins/tracksure/
```

### **2. For Developers**

```bash
# Clone repository
git clone https://github.com/your-org/tracksure.git
cd tracksure

# Install dependencies
npm install

# Build admin
cd admin
npm run build
cd ..

# Activate plugin
wp plugin activate tracksure
```

**Then**: Read [JUNIOR_DEVELOPER_GUIDE.md](JUNIOR_DEVELOPER_GUIDE.md) for next steps.
